# AWS CI/CD Web Application Deployment

## Project Overview

This project demonstrates automated deployment of a simple web application using AWS CI/CD services.
The application is deployed through a pipeline and can be accessed directly via a web browser.

## Services Used

* AWS CodePipeline — for automation of the deployment process
* AWS CodeCommit — for source code storage
* AWS CodeBuild — for building the application
* AWS CodeDeploy — for deploying to the server

## Deployment Process

1. Source code is stored in CodeCommit.
2. CodePipeline automatically triggers the build and deployment process.
3. The application is deployed to the server.
4. The deployed application is accessible through a web browser.

## Application Access

The web application can be accessed using the public URL of the deployed server in any browser.

## Outcome

This project shows a fully automated CI/CD workflow using AWS services, enabling seamless deployment and browser access.
